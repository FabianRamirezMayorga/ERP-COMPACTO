# Arquitectura ERP-COMPACTO

## Patrón: Modular Monolith → Microservicios

El proyecto inicia como un monolito modular y está diseñado para migrar a microservicios según crezca la demanda.

## Módulos

- **POS**: Punto de Venta
- **Commerce**: Gestión comercial e inventario
- **Security**: Autenticación, autorización y auditoría
