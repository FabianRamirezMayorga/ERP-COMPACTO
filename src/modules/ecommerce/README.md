# Módulo eCommerce

Módulo de tienda en línea integrado al ERP-COMPACTO.

## Funcionalidades

- 🛍️ Catálogo de productos en línea
- 🛒 Carrito de compras
- 💳 Pasarela de pagos
- 📦 Gestión de pedidos y envíos
- ⭐ Reseñas y valoraciones de productos
- 🏷️ Cupones y descuentos
- 📊 Dashboard de ventas online

## Integración con otros módulos

| Módulo | Integración |
|---|---|
| Comercio | Sincronización de inventario y productos |
| POS | Unificación de ventas online/offline |
| Seguridad | Autenticación de clientes y roles |
